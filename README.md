## Installation

To set up the project locally, follow these steps:

1. **Clone the repository**:
   
   git clone https://github.com/Dheeru12503/Task-manager.git
   cd Task-manager
2. **install dependencies** :

   npm install

## running-the-project

1. **Start the development server**
    
    npm start
